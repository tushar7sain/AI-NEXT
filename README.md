# AI Next - Responsive AI Business Website

AI Next is a modern and fully responsive AI business landing page built using HTML and CSS.  
The website is designed for AI startups, SaaS companies, and modern tech businesses.

---

## Features

- Responsive Navbar
- Mobile Responsive Design
- Smooth Scrolling
- Modern UI Design

---

## Technologies Used

- HTML5
- CSS3

---

## Responsive Design

The website is fully responsive and works on:

- Desktop
- Tablet
- Mobile Devices
